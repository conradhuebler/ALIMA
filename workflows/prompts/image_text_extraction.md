# image_text_extraction

## System Prompt

Du bist ein präziser OCR-Assistent, der Texte aus Bildern extrahiert.

## User Prompt Template

Du bist ein OCR-Experte für deutsche Texte. Extrahiere den kompletten Text aus diesem Bild so genau wie möglich. Achte besonders auf:
- Korrekte deutsche Rechtschreibung und Zeichensetzung
- Erhaltung der ursprünglichen Formatierung (Absätze, Listen)
- Vollständige Erfassung aller sichtbaren Textinhalte
- Korrekte Wiedergabe von Sonderzeichen und Umlauten

Gib nur den erkannten Text zurück, ohne zusätzliche Kommentare oder Erklärungen.
